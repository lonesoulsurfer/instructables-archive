# Instructables Archive

A personal project index and offline archive for all your Instructables projects, hosted free on GitHub Pages.

**Live demo:** [lonesoulsurfer.github.io](https://lonesoulsurfer.github.io)

---

## What you get

- A searchable, filterable website listing all your Instructables
- A full offline archive of every project - text, images, and PDFs
- Lightbox image zoom, parts list links, and PDF download buttons
- Hosted free on GitHub Pages

---

## Quick start

### 1. Create your GitHub repo

Create a new repo named exactly `YOUR_GITHUB_USERNAME.github.io`

Then enable GitHub Pages:
- Go to repo **Settings → Pages**
- Source: **Deploy from branch**
- Branch: **main**, folder: **/ (root)**

### 2. Clone the repo and add these files

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_GITHUB_USERNAME.github.io
cd YOUR_GITHUB_USERNAME.github.io
```

Download this project as a ZIP, extract it, and copy all files into your repo folder.

### 3. Install dependencies

```bash
pip install requests beautifulsoup4 playwright
python -m playwright install chromium
```

### 4. Run setup

```bash
python setup.py
```

This asks for your Instructables username and writes `config.py`. That's the only file you need to edit.

### 5. Scrape and build

```bash
python scraper/scrape.py
python scraper/download_archive.py
python build_html.py
```

The first full archive takes 1-2 hours depending on how many projects you have.

### 6. Push to GitHub

```bash
git add .
git commit -m "initial archive"
git push
```

Your site will be live at `https://YOUR_GITHUB_USERNAME.github.io` within a minute.

---

## Keeping it current

After publishing a new Instructable, run `update-site.bat` (Windows) or:

```bash
python scraper/scrape.py
python scraper/download_archive.py --new-only
python build_html.py
git add .
git commit -m "add: new project"
git push
```

## File structure

```
your-repo/
├── config.py              ← Your settings (username, display name)
├── setup.py               ← First-time setup wizard
├── build_html.py          ← Builds HTML pages from archive
├── check_archive.py       ← Checks archive completeness
├── update-site.bat        ← Windows one-click updater
├── index.html             ← Main website (reads projects.json)
├── projects.json          ← Generated by scraper
├── scraper/
│   ├── scrape.py          ← Scrapes your profile → projects.json
│   └── download_archive.py ← Downloads all projects as markdown + images
└── archive/
    └── instructables/
        ├── project-name/
        │   ├── index.md   ← Project content
        │   ├── index.html ← Built HTML page
        │   ├── images/    ← Downloaded images
        │   └── pdfs/      ← Downloaded PDFs
        └── ...
```

## Requirements

- Python 3.8+
- `requests`, `beautifulsoup4`, `playwright` (pip install)
- Git + GitHub account
- ~2GB disk space for a full archive of 200+ projects
